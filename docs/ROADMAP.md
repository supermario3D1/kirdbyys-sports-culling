# Kirdbyys Roadmap

## Version 1.0 (Current)
- [x] Project creation and folder import
- [x] YOLOv8 object detection (players, ball, goalposts)
- [x] Technical quality analysis (sharpness, exposure, noise, etc.)
- [x] Composition analysis (rule of thirds, framing, distractions, etc.)
- [x] Action & storytelling analysis for soccer
- [x] Weighted configurable ranking
- [x] Perceptual-hash duplicate and burst detection
- [x] Top N selection (10, 20, 50, 100, custom)
- [x] CSV, Excel, XMP, PDF export
- [x] Dark/light professional UI
- [x] AMD/ONNX optimized with CPU fallback
- [x] SQLite metadata and caching
